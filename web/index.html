IMAP Sync

This service runs imapsync in a container. The website only confirms the deployment is up.

To run a migration, use docker exec (or docker run) with your host, users, and passwords — not this URL.

Example:

docker exec -it <container> /usr/bin/imapsync \
  --host1 imap.source.example --user1 you --password1 secret \
  --host2 imap.dest.example --user2 you --password2 secret

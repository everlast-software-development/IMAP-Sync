1. Easiest: Coolify Terminal

In Coolify, open this resource → Terminal, then run imapsync directly (no docker exec needed):

/usr/bin/imapsync \
  --host1 imap.source.example --user1 you --password1 'your-secret' \
  --host2 imap.dest.example --user2 you --password2 'your-secret'

Add options as needed (SSL, folders, dry-run, etc.).
